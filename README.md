CryptoChecker Telegram Bot
CryptoChecker is a Telegram bot that allows users to retrieve real-time cryptocurrency prices, market data, and sentiment analysis through a simple /crypto command. The bot fetches data from the CoinGecko API, generates a visually appealing image displaying the cryptocurrency's price, market cap, volume, circulating supply, and sentiment. The sentiment is analyzed using TextBlob, based on a placeholder news sentence related to the cryptocurrency.

Features
Real-Time Cryptocurrency Price: Fetches the latest price of any cryptocurrency in USD.
Market Data: Displays additional information such as market cap, 24-hour trading volume, and circulating supply.
Sentiment Analysis: Analyzes the sentiment for the cryptocurrency based on placeholder news or discussions and displays it as positive, neutral, or negative.
Image Generation: Generates a professional-looking image that contains the crypto data and sentiment in a visually appealing format.
Simple to Use: Easy-to-use /crypto command to get information on any cryptocurrency supported by CoinGecko.
Components
1. Bot Script (bot.py)
The main script that powers the Telegram bot. It listens for commands and responds with an image that includes the cryptocurrency data and sentiment analysis.

2. Image Generation (image_generator.py)
This script handles the generation of the image. It creates a visually appealing design using the data fetched from CoinGecko and sentiment analysis from TextBlob.

3. Sentiment Analysis
Sentiment is calculated using TextBlob, a Python library that performs sentiment analysis on text data. A placeholder sentence about the cryptocurrency is used for sentiment analysis.

4. Configuration (config.py)
This file contains your bot's token (TELEGRAM_BOT_TOKEN) and other configuration details.

Installation Instructions
Prerequisites
Python 3.x
Telegram Bot Token: You'll need to create a Telegram bot via the BotFather and get your bot token.
Python Libraries:
requests for making HTTP requests to CoinGecko API
python-telegram-bot for creating the Telegram bot
textblob for sentiment analysis
Pillow for image generation

How to Use
Start the Bot: Open your Telegram app and search for your bot (use the username you set when creating the bot). Click "Start" to initiate the conversation.
Fetch Crypto Information: Type /crypto <crypto_name> to get information on any cryptocurrency. For example, use /crypto bitcoin to fetch Bitcoin data.

Code Breakdown
1. bot.py
The main logic of the bot is handled in bot.py. It uses the python-telegram-bot library to listen for the /crypto command and sends the generated image to the user.

Key Functions:

get_crypto_data(crypto_name): Fetches data from CoinGecko API.
analyze_sentiment(text): Analyzes sentiment using TextBlob.
crypto(update, context): Handles the /crypto command.
2. image_generator.py
The image_generator.py script generates the image that displays cryptocurrency data and sentiment. It uses the Pillow library to create an image and add text to it.

Key Functions:

generate_crypto_image(crypto_name, price, market_cap, volume_24h, circulating_supply, sentiment): Generates an image with the provided data and sentiment.
